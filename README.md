<h1>Sobre o Desafio</h1>
<p>Construir esta seção de cards e tornar possivel a visualização do layout ideal do site  independente do tamanho da tela do dispositivo que está sendo acessado.</p>

<h2>Minha experiência</h2>
<p>Atualmente eu parei um pouco pra revisar os meus projetos antigos e escrever um readme decente, não digo que vou fazer isso para todos, mas acho que para aqueles que causaram um plot-twist na minha cabeça eu com toda certeza farei. O lance desse desafio foi que desde que eu comecei a tentar fazer um desafio por semana, eu sempre descobria coisas novas, dentre elas o poder do FlexBox e do Grid Layout, e esse que é o lance, aqui foi a primeira vez que eu usei o Grid, mas não de forma responsiva com minmax e etc, aqui eu fiz usando media queries, porque na epoca era o mais o lance mais responsivo que eu conhecia, e pra mim era um máximo usar media querires porque dava uma sensação de "putz esse ficou bom em" entre tantos ajustes necessários para que todo aquele conteudo, ficasse alinhado de forma correta em cada dispositivo.</p> 

<h2>Sobre os desafios da FrontEnd Mentor</h2>
